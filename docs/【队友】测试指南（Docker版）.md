# AI财报分析助手 - 测试指南（Docker版）

> **测试人员操作手册** —— 使用Docker环境完成自动化测试和手工测试

---

## 一、准备工作

### 1. 安装Docker Desktop
- 下载地址：https://www.docker.com/products/docker-desktop
- 安装后重启电脑，确保系统托盘有鲸鱼图标

### 2. 获取项目代码
```bash
git clone https://github.com/Jerry518520/financial-analysis-AI-assistant.git
cd financial-analysis-AI-assistant
```

### 3. 确认环境文件
- 项目根目录应有 `.env` 文件（找负责人要）

---

## 二、启动服务（3步）

### 第1步：构建镜像（只需一次，约10-20分钟）
```bash
docker compose build --no-cache
```

### 第2步：启动容器
```bash
docker compose up -d
```

### 第3步：验证服务
- 浏览器打开 http://localhost:8501 ，应看到前端界面
- 浏览器打开 http://localhost:8000 ，应返回 `{"status": "ok"}`

---

## 三、自动化测试（168项）

**另开命令行窗口**，进入项目目录执行：

```bash
docker compose exec backend poetry run pytest tests/ -v
```

### 预期结果
```
============================== 168 passed in 25s ==============================
```

**看到 `168 passed` 即全部通过 ✅**

### 常见问题
| 情况 | 处理 |
|------|------|
| 出现 `FAILED` | 截图完整报错发给负责人 |
| 出现 `SKIPPED` | **正常**，某些库Docker里不可用，忽略即可 |

## 五、服务管理

```bash
# 停止服务
docker compose stop

# 停止并删除容器
docker compose down

# 重新启动
docker compose up -d

# 查看日志（排查问题）
docker compose logs backend
docker compose logs frontend
```
