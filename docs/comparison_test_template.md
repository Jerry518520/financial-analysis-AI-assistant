# 财报分析对比测试模板

## 测试准备

### 1. 准备测试 PDF
准备一份真实的上市公司财报 PDF，建议选择：
- 年报（信息量更大）
- 100页以内（测试时间适中）
- 包含完整财务报表

### 2. 测试渠道
| 渠道 | 说明 |
|------|------|
| 你的项目 | 启动 Docker 后访问 http://localhost:8501 |
| Seed2.0 Pro (豆包)| https://www.seed.com 或对应网页版 |
| MiniMax-M2.5 | https://chat.MiniMax-M2.5.com |

### 3. 测试提示词（统一使用）

```
你是一位专业的财务分析师。请仔细阅读这份财报（用户将上传PDF），并准确回答以下问题。

注意：
1. 如果无法从PDF中找到具体数据，请明确说明"未找到"
2. 如果需要计算，请给出计算过程
3. 回答要简洁准确，用数据说话
```

## 评分标准

### 评分维度（满分100分）

| 维度 | 权重 | 说明 |
|------|------|------|
| 数值准确度 | 30% | 数字是否正确提取/计算 |
| 计算准确度 | 20% | 计算过程和结果是否正确 |
| 分析完整性 | 20% | 是否覆盖所有问点 |
| 逻辑清晰度 | 15% | 回答是否条理清晰 |
| 回答速度 | 15% | 响应时间（主观评价） |

### 星级标准

| 星级 | 分数 | 标准 |
|------|------|------|
| ⭐⭐⭐⭐⭐ | 90-100 | 完美回答，数据准确 |
| ⭐⭐⭐⭐ | 75-89 | 正确，少量缺失 |
| ⭐⭐⭐ | 60-74 | 基本正确，有错误 |
| ⭐⭐ | 40-59 | 部分正确，较多错误 |
| ⭐ | 0-39 | 错误或无法回答 |

## 使用说明

1. **测试前**：打印或复制"记录表"到表格软件
2. **测试中**：每个问题测试3个渠道，记录原始答案
3. **测试后**：根据评分标准打分，填写评分汇总
4. **写报告**：参考"测试报告模板"整理成最终报告
