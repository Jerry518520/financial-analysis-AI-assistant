# 2026-03-28 测试文档

## 一、测试环境搭建

### 1. 环境要求

| # | 项目 | 要求 |
|---|------|------|
| 1.1 | Python | 3.11 ~ 3.14 |
| 1.2 | Poetry | 最新版（包管理工具） |
| 1.3 | Git | 用于克隆代码仓库 |
| 1.4 | 操作系统 | Windows / macOS / Linux 均可 |

### 2. 安装步骤

> 以下操作在命令行（终端）中执行，**不需要启动后端服务**。

```bash
# 第一步：克隆项目代码
git clone https://github.com/Jerry518520/financial-analysis-AI-assistant.git
cd financial-analysis-AI-assistant

# 第二步：安装项目依赖（包含测试框架 pytest）
poetry install

# 第三步：确认 pytest 安装成功
poetry run pytest --version
# 预期输出：pytest 8.x.x
```

### 3. 常见问题

| # | 问题 | 解决方法 |
|---|------|----------|
| 1.5 | `poetry` 命令不存在 | 先安装 Poetry：https://python-poetry.org/docs/#installation |
| 1.6 | `import torch` 报错崩溃 | torch 版本不兼容，执行：`pip install torchvision==0.22.0+cpu -f https://download.pytorch.org/whl/torch_stable.html` |
| 1.7 | 依赖安装超时 | Poetry 使用阿里云镜像，如仍超时可切换网络重试 |
